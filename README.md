# PayChain Plaza

A decentralized e-commerce platform powered by Solana.

## Features

- User Profile Management
- Product Listings
- Smart Cart System
- Order Tracking
- Review System
- Smart Contract Transactions

## Setup & Run

1. Install dependencies: `npm install`
2. Start the app: `npm start`
